# Socialiris Frontend

A modern, responsive "Under Construction" landing page for Socialiris, featuring a typing animation, integrated videos, and a professional layout.

## Features

- **Typing Animation**: A smooth typewriter effect for project status updates.
- **Dynamic Content**: Auto-cycling phrases with custom pauses.
- **Responsive Design**: Optimized for both desktop and mobile screens (768px breakpoint).
- **Integrated Media**: High-quality construction-themed animations.
- **Clean Structure**: Organized codebase with dedicated modules for construction alerts.

## Project Structure

- `src/construction-alert/`: Contains the core logic and styling for the landing page.
- `public/construction-alert/`: Assets used by the construction alert component.
- `index.html`: The main entry point for the application.

## Tech Stack

- [Vite](https://vitejs.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- CSS3 (Flexbox, Media Queries, Animations)

## Getting Started

### Prerequisites

- Node.js (v16.0.0 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```

### Build

Build the project for production:
```bash
npm run build
```
